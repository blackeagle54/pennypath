# PennyPath — storefront

A zero-build static landing page for **The Complete Money Bundle** (10 budgeting printables + a
budget spreadsheet). Deployed free on GitHub Pages.

## The only thing to edit

Open `index.html`, find the `window.PENNYPATH` block near the top, and paste your Gumroad product
URL into `gumroad: ""`. Every "Get the Bundle" button then points at your checkout.

```js
window.PENNYPATH = {
  gumroad: "https://pennypath.gumroad.com/l/money-bundle",
  price: "$14",
  compareAt: "$45"
};
```

## Deploy

It's plain HTML/CSS — already live via GitHub Pages from the `main` branch root. Push a change and
it redeploys in ~1 minute. To preview locally:

```bash
python3 -m http.server 8000   # then open http://localhost:8000
```

## Files

| Path | What |
|---|---|
| `index.html` | The whole page (semantic HTML, SEO + OG tags, structured data) |
| `style.css` | Brand styles (navy/teal, Inter + Playfair), responsive |
| `assets/covers/` | Product cover thumbnails |
| `assets/grid-everything.png` | "What you get" image |
| `assets/og-card.png` | Social share image |
| `favicon.svg` | Brand mark |

Assets are generated by the sibling `pennypath-products` repo (`bun run src/mockups.ts`).
